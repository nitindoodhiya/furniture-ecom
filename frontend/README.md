
# Minimal Frontend for Furniture E-commerce

How to run:

1. Copy `.env.example` to `.env.local` and set `NEXT_PUBLIC_API_URL` to your backend (default: http://localhost:4000).
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run dev server:
   ```bash
   npm run dev
   ```
4. Open http://localhost:3000
